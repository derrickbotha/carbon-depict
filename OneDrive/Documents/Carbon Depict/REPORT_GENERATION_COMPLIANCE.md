## üìù Technical Details

### API Endpoints:
```javascript
// Generate report
POST /api/reports/generate
Body: {
  framework: 'GRI',
  reportType: 'Annual',
  year: 2025,
  format: 'pdf',
  includeData: true,
  includeCharts: true
}

Response: {
  success: true,
  message: 'Report generation started',
  data: { /* report object */ }
}
```

```javascript
enterpriseAPI.reports.generate({
  framework: 'GRI',
  reportType: 'Annual',

  includeData: true,
  includeCharts: true
})
```
